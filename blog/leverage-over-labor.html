Transmission 004

Leverage Over Labor

Hard work is romanticized. But hard work alone won't make you rich, and it won't change the world. A ditch digger works harder than a software engineer, but the engineer has leverage.

Code and media are permissionless leverage. A piece of code runs while you sleep. A video or essay is consumed by thousands of people simultaneously without any marginal cost of reproduction.

If you are still trying to out-work your competition using raw, un-leveraged human hours, you have already lost. Build systems. Write code (or use tools that do). Create media. Disconnect your output from your inputs.
