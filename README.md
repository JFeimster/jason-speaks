# Jason Speaks

A neobrutalist, futurist fintech speaker profile landing page for Jason Feimster.

## Live Site

Coming soon: https://jason-speaks.vercel.app

## Purpose

This site is designed to help event organizers, conferences, corporate teams, universities, and founder communities book Jason for keynotes, panels, workshops, and high-energy talks on fintech, AI, ownership, entrepreneurship, and the new economy.
